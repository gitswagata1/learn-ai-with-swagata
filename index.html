    🪄 The AI Power-User Cheat Sheet

    🧩 The recipe — R-T-C-F
    • Role — who it acts as
    • Task — what to do
    • Context — details it needs
    • Format — how it should look
    🧠 Patterns
    • Persona · Few-shot examples
    • “Think step by step”
    • “Ask me questions first”
    • “Using only the text below…”
    • “Critique, then improve”
    💬 Magic follow-ups
    • Make it shorter / simpler.
    • Give me 3 versions.
    • What should I double-check?
    ⚡ Power features
    • 📎 Files · 👁️ Images · 🎙️ Voice
    • 🌐 Web search · 🧠 Memory
    • 🗂️ Projects · 🎨 Image gen
    ✅ Trust it for
    • Writing, summarizing, explaining
    • Brainstorming, planning, organizing
    ⚠️ Verify / protect
    • Facts, numbers, news, citations
    • Never paste secrets or private data
    🔒 Golden rule
    • Postcard test in. Verify what matters. You decide.

    Glossary — plain English

    LLM

    “Large Language Model” — the prediction engine behind chat assistants.

    Prompt

    The message/instruction you give the AI.

    Token

    A small chunk of text the model processes.

    Context window

    How much it can “hold in mind” at once.

    Hallucination

    A confident but false answer. Verify facts.

    Model

    A specific AI “brain” version. Newer ≈ smarter.

    Multimodal

    Handles more than text — images, audio, video.

    Agent

    AI that takes actions & completes multi-step tasks.

    Frequently asked

    Q Do I need to be technical?
    No. If you can send a text message, you can use AI. This whole course assumes zero background.
    Q Is it free?
    Yes — this course is free, and all major AI tools have capable free tiers you can complete it with.
    Q Why does AI sometimes get things wrong?
    It predicts plausible text rather than looking up verified facts. Superb at language, imperfect at precise facts — so verify what matters.
    Q Which tool should I pick?
    Any major one — Claude, ChatGPT, or Gemini. Get fluent in one; the skills transfer everywhere.
    Q Will my progress be saved?
    Yes — it’s saved in your browser automatically, so you can close the tab and pick up where you left off.
